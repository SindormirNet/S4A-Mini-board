S4A Mini Board
========================================

Esta mini placa está pensada para conectarse con una placa Arduino UNO o Leonardo a modo de Shield y proporciona los sensores y actuadores más habituales:

- Conectores para dos motores Servo
- Dos potenciómetros
- Dos pulsadores
- Dos LEDs digitales
- Un LED conectado a salida PWM
- Pines de 5V y GND libres
- Pines A3, A4, A5, 5, 6, 8 y 11 libres para conectar con elementos externos
